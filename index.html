About Yamsafer

Yamsafer is an online travel website that provides quick and direct hotel bookings in the Middle East & Turkey. Our hotels range from international chains to budget and boutique hotels, where we guarantee not only the best prices among our competition, but also ensure value for money paid. Our team is dedicated to a holistic transparency approach, where we are committed to providing the best prices and most up to date information when it comes to our hotel description and policies. In addition, and as a result of the successfull travel data aggregation we succeeded to acquire over time, Yamsafer extends itâ€™s data to partners through a set of RESTful Application Programming Interfaces (APIs). This documentation will explain in details how to interact with Yamsafer APIs.

API Authentication

Sign-up

To use our travel API, please contact Yamsafer Dev Team

Authentication

Upon signup, an API key and a username combination will be assigned to the organization or person handling the data request. All requests sent must contain the apiKey parameter

                        

                            

                                Name

                                Type

                                Description
                            


                            

                                apiKey

                                String

                                Verifies the identity of the request

Example

https://www.yamsafer.me/api/v2/hotels?API_KEY=uAo5kiMyvaWOByceXXDm1dJ6Hx9

Hotels

Hotels List

Returns a list of all hotels available via Yamsafer and other business partners

Example

https://www.yamsafer.me/api/v2/hotels?API_KEY=uAo5kiMyvayceXXDm1dJ6Hx9&lang='ar'&limit=10&page=0

Request

                        

                            

                                Parameter

                                Default

                                Required

                                Description
                            


                            

                                lang

                                "en"

                                No

                                Language of the results returned
                            

                            

                                limit

                                "10

                                No

                                Number of hotels per request
                            

                            

                                page

                                0 (zero)

                                No

                                Offset to start from equivalent to the offset query 
parameter in SQL This parameter will decide at which hotel to start.

Response

Returns a list of hotel objects

                        

                            

                                Name

                                Type

                                Description
                            


                            

                                hotel_id

                                Integer

                                ID for the property. This same ID will be used in any subsequent 
availability requests or hotel info		
                            

                            

                                hotel_name

                                String

                                Name of the property

                            

                            

                                description

                                String

                                Description text entered by the property

                            

                            

                                stars

                                Float

                                Star rating of the hotel, a value of 0.0 or blank indicate none is available

                            

                            

                                longitude

                                Float

                                Longitude coordinates for the hotel

                            

                            

                                latitude

                                Float

                                Latitude coordinates for the hotel

                            

                            

                                image

                                String

                                URL for the hotel cover image

                            

                            

                                address

                                String

                                Hotel street address

                            

                            

                                country

                                String

                                The country where the hotel is located

                            

                            

                                city

                                String

                                The city where the hotel is located

                            

                            

                                is_cardless

                                Boolean

                                True, if the hotel accepts cardless reservations

                            

                            

                                pre_paid

                                Boolean

                                True, if the hotel requires payment information in order to hold a reservation

                            

                            

                                low_rate

                                String

                                Amount of average low rate for this hotel within seven days period

                            

                            

                                low_rate

                                String

                                Amount of average low rate for this hotel within seven days period

                            

                            

                                amenities

                                Array

                                Lists general amenities at the property

                            

                            

                                serp_filters

                                Array

                                Search Engine Results Page filters. List of strings used to filter results, example: has_parking, has_internet

                            

                            

                                path

                                String

                                Path to the hotel page on Yamsafer site
